## 第一步：领取你的免费 `.me` 域名

一个好记的域名是网站的门牌号。借助 GitHub 学生包，我们可以在 Namecheap 上免费“白嫖”一年的 `.me` 域名。

1.  登录你的 GitHub 账号，进入 [GitHub 学生包福利页面](https://education.github.com/pack/offers)。
2.  在优惠列表中找到 **Namecheap**，点击它提供的专属链接。你会被引导至 Namecheap 网站，并看到一个专门为 GitHub 学生提供的优惠。
3.  在页面上的搜索框里，输入你心仪的域名。比如 `hello-yourname.me`。
4.  点击搜索，如果域名未被注册，你会看到一个价格为 $0.00 的选项。把它加入购物车，然后完成结算流程。
    *   *提示：结算过程中 Namecheap 会要求你注册或登录账号，并填写一些个人信息，照实填写即可。整个过程不需要任何支付信息。*
5.  完成之后，你就拥有了这个域名一年的使用权！你可以在 Namecheap 的 `Dashboard` -> `Domain List` 中看到它。
> 这个网页可以先不关闭，稍后有用

## 第二步：用 AI 生成你的第一个网页

现在我们有“门牌号”了，接下来需要“房子”。别担心，我们不需要一砖一瓦地去敲代码，让 AI 来为我们代劳。

现在，借助各种强大的 AI 工具，你只需通过文字描述，就能快速生成一个网站的完整代码。虽然市面上有许多一键式 AI 网站生成器，但在这里，我们以 Google AI Studio 为例，向你展示如何利用其强大的语言模型（如 Gemini）来生成网页代码。这种方法让你对代码有完全的控制权。
> 请注意，目前的 AI 技术虽然强大，但仍然只基本支持生成精美的单页静态页面，如果你需要更复杂的功能/多页应用，可能还需要更进阶的工具  
> 静态页面指的是那些不需要服务器端处理的网页，所有内容都直接存储在 HTML、CSS 和 JavaScript 文件中。它们适合展示个人作品集、博客或简单的业务介绍网站。

1.  **打开 Google AI Studio**: 访问 [Google AI Studio](https://aistudio.google.com/) 并使用你的 Google 账号登录。

2.  **选择或创建提示 (Prompt)**:
    *   进入后，点击左侧的 “build” 按钮创建一个新的 app 。
    *   在输入框中，你需要清晰、具体地描述你想要的网站。这不仅仅是描述外观，更要说明你需要生成的具体代码。
    *   > 对于初学者来说，我建议你参考以下示例提示，直接生成Html、CSS和JavaScript代码，而不要让AI生成框架代码（如 React、Vue 等），以免增加不必要的复杂度。
    *   **示例提示**: “`请为我创建一个个人作品集网站。我叫李华，是一名摄影师和网页开发者。请帮我生成完整的 HTML、CSS 和 JavaScript 代码。
        *   **页面结构 (HTML)**: 需要包含一个导航栏、一个展示我摄影作品的画廊区域、一个“关于我”的简介部分，以及一个包含社交媒体链接的页脚。
        *   **网站风格 (CSS)**: 整体采用简约、现代的设计风格，使用深色主题。画廊的图片要有悬停放大效果。字体请使用无衬线字体。
        *   **网站功能 (JavaScript)**: 导航栏在页面向下滚动时，背景变为半透明。画廊区域可以点击左右箭头切换图片。`”

3.  **生成、审查和复制代码**:
    *   AI 会根据你的描述生成代码。在页面的右侧你可以预览各个平台设备生成的效果。
    *   仔细检查生成的效果是否符合你的预期。如果不满意，你可以修改提示，或在后续对话中要求 AI 进行调整，例如：“`请把背景颜色改得再深一点`” 或 “`给图片画廊增加一个响应式布局，使其在手机上也能正常显示`”。

4.  **保存为本地文件**:
    *  当你觉得满意时，点击右上角的下载按钮，将生成的代码导出为一个 `.zip` 文件。

5.  **本地预览和修改**:
    *  解压下载的 `.zip` 文件到你电脑上的一个文件夹中。
    *  在浏览器中直接打开 `index.html` 文件，你就可以看到你的网站了。
    *  如果你想对网站进行进一步的个性化修改（例如更改网站的标题等），可以使用任何文本编辑器（如 VSCode、Notepad++ 等）打开 HTML、CSS 和 JavaScript 文件，进行编辑。

## 第三步：在 Cloudflare 上发布你的网站

有了网站文件，我们还需要一个地方来存放它，并让全世界的人都能访问。Cloudflare Pages 就是一个完美的免费选择。

1.  用你的邮箱注册并登录 [**Cloudflare** 仪表盘](https://dash.cloudflare.com/)。
2.  在左侧侧边栏找到 **Compute & AI** -> **Workers & Pages**，点击进入。
3.  选择 **Create application** -> **Pages** -> **Create application** ->**Upload your static files**。
4.  **给你的项目起个名字**，比如就叫 `my-first-website`。
5.  **上传你的网站**：将你刚刚解压出来的文件夹直接拖拽到上传区域。
6.  点击 **Deploy site**。

稍等片刻，Cloudflare 就会完成部署。完成后，你会得到一个 `*.pages.dev` 结尾的网址（例如 `my-first-website.pages.dev`）。点开它，恭喜你，你的网站已经成功上线了！

## 第四步：绑定你的专属域名

虽然 `pages.dev` 的域名也能访问，但使用我们自己的 `.me` 域名才算真正拥有一个“专属”网站。

1.  在刚刚部署成功的 Cloudflare Pages 项目页面，点击 **Custom domains** -> **Set up a custom domain**。
2.  输入你从 Namecheap 获取的那个 `.me` 域名，然后点击 **Continue**。
3.  Cloudflare 会开始验证。由于你的域名当前由 Namecheap 管理，Cloudflare 会提示你需要更改域名的 **Nameservers (域名服务器)**。它会提供两个新的 Nameserver 地址，看起来像这样：
    *   `luke.ns.cloudflare.com`
    *   `leia.ns.cloudflare.com`
    *   *请复制好这两个地址，你自己的可能会和这个不一样。*
4.  **回到 Namecheap**：
    *   进入你的 [Namecheap](https://ap.www.namecheap.com/) `Dashboard` -> `Domain List`。
    *   找到你的 `.me` 域名，点击旁边的 `Manage` 按钮。
    *   在 `NAMESERVERS` 区域，你会看到一个下拉菜单，默认是 `Namecheap BasicDNS`。点击它，选择 `Custom DNS`。
    *   将 Cloudflare 提供的那两个 Nameserver 地址粘贴到输入框中，然后点击绿色的对勾保存。
5.  **等待生效**：现在，你需要做的就是耐心等待。域名服务器的变更通常需要几分钟到几小时不等才能在全球生效。
6.  你可以回到 Cloudflare 页面，它会自动检测状态。当它显示 `Active` 状态时，就大功告成了！Cloudflare 会自动为你的网站配置好 SSL 证书，让你的网站可以通过 `https://` 安全访问。
